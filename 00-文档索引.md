## 📂 项目结构

```
weiqi-go/
├── 📚 核心文档（根目录）
│   ├── 00-系列：规则和API

│   └── 10-21系列：功能模块
│
├── 📁 docs/
│   └── bug-fixes/         # Bug修复记录
│       ├── 修复记录汇总.md
│       └── archive/       # 详细文档归档
│
├── 🔧 后端代码
│   ├── game/          # 游戏引擎
│   ├── api/           # API服务
│   ├── auth/          # 认证系统
│   ├── user/          # 用户管理
│   ├── storage/       # 数据存储
│   ├── database/      # 数据库
│   ├── logger/        # 日志系统
│   ├── ai/            # AI客户端
│   └── config/        # 配置管理
│
├── 🤖 AI服务
│   └── weiqi-ai/      # Python AI服务
│
├── 🎨 前端代码
│   └── weiqi-frontend/  # Vue 3前端
│
└── 🐳 部署配置
    ├── Dockerfile
    ├── docker-compose.yml
    └── Makefile
```
