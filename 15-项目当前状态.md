## 🎯 项目概况

Weiqi-Go 是一个功能完整的在线围棋游戏平台，采用 Go + Python 微服务架构，支持玩家对战和人机对弈。

## 🚀 快速开始

### 后端服务（已可用）

```bash
cd /home/zhuji/weiqi-go

# 一键部署
make deploy

# 查看日志
make logs

# 测试 API
make test-api
```

访问：
- 后端 API: http://localhost:8080
- AI 服务: http://localhost:8000
- AI 文档: http://localhost:8000/docs

### 前端开发（需要 Node.js）

```bash
# 1. 安装 Node.js
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
source ~/.bashrc
nvm install 20
nvm use 20

# 2. 安装依赖
cd /home/zhuji/weiqi-go/weiqi-frontend
npm install

# 3. 启动开发服务器
npm run dev
```

访问: http://localhost:3000
